# vue多页应用命令行工具
<br/>

# 使用以下方法安装
```
npm install -g webpack webpack-dev-server git+https://github.com/u-san/usan-cli.git
```


# command
```
qy         # 显示可用command
qy init    # 初始化项目目录（默认为fe)
qy start   # 执行菜单列表 
qy upgrade # 更新工具
```

# 菜单列表
* 安装依赖
* 更新依赖
* 新建页面
* 启动开发服务器
* 使用调试模式
* 编译全部
* 更新模板库
* 回到初始菜单
